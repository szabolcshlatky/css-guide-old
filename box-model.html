CSS Box Model

display
inline // Can't be positioned and box modelled, but it takes only content-size of the element.
block // Can be positioned and box modelled, but takes a full width of the page.
inline-block // Can be positioned, box modelled and other (non-block level) elements can be next to it.
none vs. visibility: hidden // The visibility keeps the unseen element's position.
position
static // natural HTML flow. z-index doesn't work with it.
relative // to its static HTML position.
absolute // relative to its (relative) parent element and takes out the element from the natural HTML flow.
fixed // relative to its viewport.
Centering elements
text-align: center; centers a text inside a block, but not centers the block itself.
margin: 0 auto; centers a block (with width less than 100% or auto) horizontally.
Inside a relative positioned element... 
      {
        position: absolute; // or fixed (relative positioned parent doesn't needed)
        top: 50%; // or bottom: 50%
        left: 50%; // or right: 50%
        transform: translate(-50%, -50%);
      }
