    Text

    A visually impaired user might want to make the text bigger on all websites they visit, or a user with severecolor deficiency might want to put all websites in high contrast colors that are easy for them to see. Whatever the need, you should be comfortable with this, and make your designs flexible enough so that such changes will work in your design. As an example, you might want to make sure your main content area can handle bigger text (maybe it will start to scroll to allow it all to be seen), and won't just hide it, or break completely.

    Link styling

    Something should definitely happen when states change, and you shouldn't get rid of the pointer cursor or the underline — both are very important accessibility aids for those using keyboard controls. Color should not be used as the sole method of distinguishing links from non-linking content. Link text color, like all text, has to be significantly different from the background color (a 4.5:1 contrast). In addition, links should visually be significantly different from non-linking text, with a minimum contrast requirement of 3:1 between link text and surrounding text and between default, visited, and focus/active states and a 4:5 contrast between all those state colors and the background color.

    Form styling

    You shouldn't however deviate too much from the expected visual feedback form elements receive when they are focused, which is basically the same as links (see above). You could style form focus/hover states to make this behavior more consistent across browsers or fit in better with your page design, but don't get rid of it altogether — again, people rely on these clues to help them know what is going on.

    Choose colors suitable for the page theme!
    Red: Passion, energy, intensity - For cars
    Yellow: Joy, intellect, attention - For advertising that gets noticed or for maximum navigation.
    Green: Freshness, safety, growth - For food
    Blue: Stability, trust, calmness - Money themes, Ictys
    Purple: Superiority, wealth, femininity - Themes aimed at women
    Choose 2-3 colors (+ black, white)!
    Analogous choice: several shades of a color (e.g. red, orange, burgundy) - It can be good for navigation, text body, logo and its background.
    Complementary choice (recommended - except for text and background): Opposite colors
    • Dark Blue - Mustard Yellow
    • Blue - Orange
    • Light Blue - Dark Orange
    • Green - Red
    • Light green - Burgundy
    • Lemon Yellow - Purple
    Dragging shapes: ...Color circle
    Color palette generation: Adobe Color
    AAA contrasts: Your text color should contrast well with your background color.
    For typography use maximum 2 font families!
    Title: serif OR contrastingly fatter sans-serif
    Text body: 'humanist' sans-serif
    • Serif: Traditional, stable, respectable
    • Sans-serif: Sensitive, simple, clear
    • Cursive: Personal, creative, elegant
    • Display: Friendly, loud, fun
    • Modern: Stylish, fashionable, smart
    Select sensible font sizes, line heights, letter spacing, etc. to make your text logical, legible, and comfortable to read.
    Make sure your headings stand out from your body text, typically big and bold like the default styling. Your lists should look like lists.
    Spacing may be created using CSS properties such as margin.

    Tables: Table CSS generally serves to make the table fit better into your design and look less ugly. It is a good idea to make sure the table headers stand out (normally using bold), and use zebra striping to make different rows easier to parse.
